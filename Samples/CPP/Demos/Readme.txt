For all this demos, it supposed that all CPP examples are 
in the ..\GLScene\CPP\Demos  directory
1- the header files are in ..\GLScene\include
2- the lib files are in ..\GLScene\lib
3- the media files are in ..\GLScene\Demos\media

Please, for your own projects, edit the options and correct the include and lib path 
according to your configuration.

Additional Dynamic libraries, dll with Setup.bat are in ..\GLScene\Source\external directory
