For all this demos, it supposed that all CPP examples are 
in the D:\Library\GLScene\CPP\Demos\ directory
and GLScene are in D:\Library\GLScene\
1- the header files are in D:\Library\GLScene\include
2- the lib files are in D:\Library\GLScene\lib
3- the dcu/obj files are in D:\Library\GLScene\dcu
4- the media files are in D:\Library\GLScene\Demos\Media

Please, for each project, edit the options and correct the include and lib path 
according to your configuration.

In some examples you need to have access to models, maps, skins and textures in 
D:\Library\GLScene\CPP\Demos\media directory

Additional Dynamic libraries, dll with bat installer are in _Install_DLLs directory
