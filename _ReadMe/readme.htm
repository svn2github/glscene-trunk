GLScene
OpenGL Solution for Delphi, C++Builder and Lazarus

maintained by
Eric Grange

and GLS Team, based on Mike Lischke's GLScene

GLScene is an OpenGL based 3D library for Delphi, C++Builder and Lazarus. It provides visual components and objects allowing description and rendering of 3D scenes in an easy, no-hassle, yet powerfull manner.

GLScene is not just an OpenGL wrapper or utility library, it has grown to become a set of founding classes for a generic 3D engine with Rapid Application Development in mind. GLScene allows you to quickly design and render 3D scenes without having to learn the intricacies of OpenGL: if you know how to design a TForm, you'll easily master the basic operation of TGLScene. Make sure to checkout the introduction tutorial included in the help folder.
GLScene includes more than many components and a variety of founding classes. The library comes with a comfortable set of demos showcasing the ease of use and demonstrating RAD wasn't done at the expense of CPU horsepower. Advanced demos and samples can be found in the project repository at http://sourceforge/projects/glscene/

The GLScene Sound System allows positionnal 3D audio via low-level APIs like FMOD and BASS.

Installing GLScene :

  • Use screenshots from GLScene\_Installation directory
  • Setup Environment Options for Delphi and C++Builder compilers as shown in an appropriate screenshot dialogs.
  • Create a project group GLScene and add existing *.dpk projects with right order as in ProjectManager_GLScene_groupproj.jpg
  • Setup Delphi compiler for all *.dpk projects to generate C++Builder files and packages as shown in ProjectOptions_DelphiCompiler_OutputCPP.jpg
  • Setup all necessary updated DLLs using bat file from ..\Source\external directory.
  • Build the list of projects and install GLScene’s components from DesignTime packages.
  • For sound support via FMOD/BASS/OpenAL and Phisics also install appropriate .dpk packages.

Up-to-date compiling instructions and details may be found in the "readme.htm" of the "Sources" directory.

GLScene supports Delphi 7 and previous.

If you installed a previous version, make sure no files of the previous version remains, some file may have been renamed, moved or removed in between versions. Use MassReplace utility to update old versions to current one.
