Installation of GLScene
into Embarcadero RAD Studio

The Instruction for installing GLScene library into Embarcadero RAD Studio XE5 to work with Delphi & C++Builder.

1. Download GLScene from the site http://sourceforge.net/projects/glscene into directory D:\Library\GLSceneSVN.

Archives of all GLScene’s versions are on the page http://sourceforge.net/projects/glscene/files/

The last release of GLScene could be find in  http://sourceforge.net/projects/glscene/files/GLScene/

In addition, you could checkout the current SVN version of GLScene using TortoiseSVN client, where should be inserted the next url:  http://svn.code.sf.net/p/glscene/code/trunk

 

2.Make a copy of GLScene’s trunk in a separate directory, e.g. in the folder D:\Library\GLScene

 

3. Setup auxiliary dynamic libraries for 3D sound, shaders and physics (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton) in the directory C:\Windows\System32. Please, run setup.bat in D:\Library\GLScene\Source\external directory. Note: when using 64-bit OS Windows you may need to copy appropriate *.dll  files in C:\Windows\SysWOW64 directory by hands.

 

4. Specify environment options of Embarcadero RAD Studio for autosave as shown in the next figure.

 

 

5. Delphi Options. Add the directory D:\Library\GLScene\Source to search source files of GLScene in the Library Path. Also make output of header files into the HPP output directory D:\Library\GLScene\Source\include.

 

 

 

6. C++ Options. Add D:\Library\GLScene\Source\include folder into Ordered list of Include paths in Paths and Directories with header hpp files:

 

 

7. C++ Options. Add D:\Library\GLScene\Source\lib folder to Ordered list of Library paths in Paths and Directories with bpi/lib files.

 

 

 

8. Open the GLSceneXE5.groupproj file that is in D:\Library\GLScene\XE5 directory by using menu item File|Open Project…(Ctrl+F11). In the Project Manager window you need to have 12 projects, including GLS_SDL.dpk. Note: you may open, compile and install every dpk project apart of the group project.

 

 

9. Be sure, that for every project in the group there are inserted right Delphi Compiler options (Projects|Options… Shift+Ctrl+F11) to output C/C++ files in the include directory (HPP files) and the lib directory (bpi/lib files),  and should be generated all C++Builder files including bpl and package libs.

 

 

 

10. Build GLScene’s packages. Right click mouse button on the GLSceneXE5 group project line in Project Manager window and then execute Build All command. After that Install visual components into RAD Studio palette for all projects, except GLS_SDL.dpk (still not upgraded for the current version).

 

The installation is finished. You may work with GLScene’s component library now. By following described steps above you can find useful samples for Delphi and C++Builder in D:\Library\GLScene\Samples directory.
