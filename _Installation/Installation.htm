Installation of GLScene
into Embarcadero RAD Studio

The Instruction for installing GLScene library into Embarcadero RAD Studio to work with Delphi & C++Builder includes next steps:

1. Download GLScene from the site http://sourceforge.net/projects/glscene to your work directory, e.g. D:\Library\GLSceneSVN.

You can download also a current or previous releases of GLScene project on the page http://sourceforge.net/projects/glscene/files/

When using TortoiseSVN client checkout the current SVN version of GLScene from the next repository url:  http://svn.code.sf.net/p/glscene/code/trunk

 

2.Make a copy of GLScene’s trunk in a separate directory, e.g. in the folder D:\Library\GLScene

 

3. Setup auxiliary dynamic libraries for 3D sound, shaders and physics (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton) in the directory C:\Windows\System32. Please, run setup.bat in D:\Library\GLScene\Source\external directory. Note: when using 64-bit OS Windows you may need to copy appropriate *.dll  files to C:\Windows\SysWOW64 directory manually.

 

4. Check Autosave options for Embarcadero RAD Studio as shown in the next figure.

 

 

5. Setup Delphi Options. Add the directory D:\Library\GLScene\Source to search source files for GLScene’s applications  in the Library Path.

 

 

 

To properly use the DesignEditors, DesignIntf, ToolsAPI and VCLEditors units you need to add the compiler option -LUDesignIDE  on the Compiling page in Project Options:

 

6. Setup C++ Options for Paths and Directories. Add D:\Library\GLScene\Source\include folder into Ordered list of Include paths to find header HPP files by GLScene’s applications:

 

 

7. Open the GLScene.groupproj file in your D:\Library\Packages\ directory using menu item File|Open Project…(Ctrl+F11). In Project Manager window you should have a list of projects with *.bpl extensions:

 

 

8. Setup Delphi Compiler options for every DesignTime project in GLScene’s project group (Projects|Options… Shift+Ctrl+F11) to get output C/C++ files into include (HPP files) and library (BPI/LIB files) directories. Choose the option “Generate all C++Builder files (including package libs) for DesignTime packages”. You can use a special file ..GLScene\Packages\CDOutput.optset to apply necessary paths to all DesignTime packages:

 

9. For RunTime packages you may leave Output C/C++ files  with default options:

 

 

10. Build all GLScene’s packages. Right click mouse button on the first line of projects from GLScene group in Project Manager window and execute Build All From Here command. Then choose to Install every DesignTime package of visual components into RAD Studio palette.

 

After installation you can start to work with GLScene’s component library and study demos for Delphi and C++Builder in ..\Samples directory.
