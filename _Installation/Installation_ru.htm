Установка GLScene
в Embarcadero RAD Studio

Инструкция по установке библиотеки GLScene в Embarcadero RAD Studio для работы в Delphi & C++Builder. Выполните следующие шаги:

1. Загрузите исходные коды библиотеки GLScene из репозитория с URL of repository https://svn.code.sf.net/p/glscene/code/trunk сайта http://sourceforge.net/projects/glscene в директорию D:\Library\GLSceneSVN на вашем диске.

Для загрузки кода (svn checkout) используйте программу TortoiseSVN или встроенную в среду RAD Studio систему поддержки версий, вызываемую из меню File | Open From Version Control…

Кроме того, со страницы http://sourceforge.net/p/glscene/code/HEAD/tree/  можно загрузить Snapshot текущей версии.

Последний релиз находится в разделе  http://sourceforge.net/projects/glscene/files/GLScene/

 

2. Скопируйте папку trunk в отдельную рабочую директорию, например, с именем D:\Library\GLScene_VCL, чтобы предотвратить случайное изменение исходных файлов. Можете пропустить этот шаг, если вы не собираетесь далее обновлять файлы с помощью системы контроля версий.

 

3. Установите в директорию Windows\System32 вспомогательные внешние динамические библиотеки 3D звука, шейдеров и физики (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton). Иначе вам придется для каждого приложения, обращающегося к этим библиотекам, помещать соответствующие DLL в директории с исполняемыми файлами.  Запустите командный файл setup.bat, находящийся в директории  D:\Library\GLScene_VCL\Source\external. Замечание: если при запуске setup64.bat на 64-х разрядных OS Windows возникнут проблемы с доступом к системным папкам, то вручную скопируйте соответствующие файлы *.dll в директорию Windows\SysWOW64.

 

 

4. Опции Delphi. Добавьте путь в строке Library Path к исходным файлам библиотеки GLScene,  расположенным в директории D:\Library\GLScene_VCL\Source:

 

 

5. Для корректного подключения модулей DesignEditors, DesignIntf, ToolsAPI и VCLEditors для последних компиляторов, как указано в справке RAD Studio,  необходимо добавить опцию Delphi  компилятора  -LUDesignIDE  на закладке Compiling в диалоге Project Options:

6. Опции C++ Builder для файлов заголовков HPP. Откройте диалог Options в меню Tools. Добавьте путь в строке System Include path в секции Paths and Directories для доступа C++ приложений GLScene к заголовочным hpp файлам, находящимся в директории D:\Library\GLScene_VCL\Source\include:

 

 

 

7. Опции C++ Builder для файлов библиотек BPI/LIB. Откройте диалог Options в меню Tools. Добавьте путь для доступа C++ приложений GLScene к библиотечным файлам  “D:\Library\GLScene_VCL\Source\lib”  в строке “Ordered list of Library paths”:

 

 

8. Откройте через меню File|Open Project…(Ctrl+F11)  групповой проект  GLScene.groupproj, находящийся в директории D:\Library\GLScene_VCL\Packages\RAD_Studio. В окне Project Manager должны быть перечислены следующие проекты с расширением bpl:

 

 

9. Установите опции компилятора Delphi для вывода C/C++ файлов в директории с HPP и BPI/LIB файлами для всех DesignTime проектов в группе GLScene (Projects|Options… Shift+Ctrl+F11). По умолчанию необходимые пути инсталляции пакетов прописаны в файлах проектов *dproj. Если они не соответствуют расположению ваших папок, то установите в опциях компилятора Delphi (Projects|Options… Shift+Ctrl+F11) для каждого проекта вывод файлов C/C++ в соответствующие директории include с файлами HPP и lib с файлами bpi/lib.  Выберите опцию Generate all C++Builder files (including package libs) -  сгенерировать все C++Builder файлы (включая библиотеки libs). Можно использовать специальный файл  ..Packages\CPPoutput.optset для применения готовых установок путей, загрузив его с помощью кнопки Apply… в диалоге опций проекта:

 

 

10. Откомпилируйте все пакеты группового проекта GLScene. Для этого щелкните правой кнопкой мыши в окне Project Manager по верхней строке проекта GLScene и выполните команду  Compile All From Here.  После компиляции установите пакеты с визуальными компонентами в палитру компонентов RAD Studio с помощью команды Install.

(Замечание: пакет GLS_SDL_VCL.dpk может быть откомпилирован и использоваться в режиме run time, но конфликтует в новой средой разработки в режиме design time, поэтому не устанавливайте его в палитру компонентов, пакеты GLS_DWS_VCL GLS_Python_VCL требуют подключения вспомогательных библиотек, которые в данную версию GLScene не включены).

 

Установка пакетов компонентов GLScene завершена. Далее можно приступить к работе с библиотекой и выполнить  примеры для Delphi и C++Builder, находящиеся в директории ..GLScene_VCL\Samples.
