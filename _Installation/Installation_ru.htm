Установка GLScene
в Embarcadero RAD Studio

Инструкция по установке библиотеки GLScene в Embarcadero RAD Studio XE5 для работы в Delphi & C++Builder.

1. Загрузите GLScene с сайта http://sourceforge.net/projects/glscene в директорию D:\Library\GLSceneSVN.

Архивы всех версий GLScene находятся на странице http://sourceforge.net/projects/glscene/files/

Последний релиз находится в разделе  http://sourceforge.net/projects/glscene/files/GLScene/

Кроме того, текущая версия разработки GLScene может быть загружена (svn checkout) с помощью программы работы с версиями TortoiseSVN, в которой должен быть указан следующий url адрес:  http://svn.code.sf.net/p/glscene/code/trunk

 

2. Сделайте копию содержимого папки trunk библиотеки GLScene в отдельной рабочей директории, например, в папке D:\Library\GLScene

 

3. Установите вспомогательные внешние динамические библиотеки 3D звука, шейдеров и физики (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton) в директорию Windows\System32.. Для этого запустите командный файл setup.bat, находящийся в директории  D:\Library\GLScene\Source\external. Замечание: при использовании 64-х разрядной OS Windows может потребоваться вручную скопировать соответствующие файлы *.dll в . Windows\SysWOW64.

 

4. Настройте опции среды разработки Embarcadero RAD Studio для автосохранения как показано на следующем рисунке.

 

 

5. Опции Delphi. Добавьте путь, Library Path,  к исходным файлам библиотеки GLScene, расположенным в директории D:\Library\GLScene\Source. Укажите вывод заголовочных файлов (HPP output directory) в папку D:\Library\GLScene\Source\include.

 

 

 

6. Опции C++. Добавьте путь, System Include path в секции Paths and Directories, к заголовочным файлам, находящимся в директории D:\Library\GLScene\Source\include:

 

 

7. Опции C++. Добавьте путь, System Include path в секции Paths and Directories, к библиотекам bpi/lib, находящимся в директории D:\Library\GLScene\Source\lib.

 

 

 

8. Откройте через меню File|Open Project(Ctrl+F11)  групповой проект  GLSceneXE5.groupproj, находящийся в директории D:\Library\GLScene\XE5. В окне Project Manager должно быть 12 проектов, включая GLS_SDL.bpl. Замечание: вы можете открыть, скомпилировать и установить каждый из проектов dpk отдельно без использования группового проекта.

 

 

9. Убедитесь, что в опциях компилятора Delphi (Projects|Options Shift+Ctrl+F11) для каждого проекта из группы при выводе файлов C/C++ указаны правильные пути к директориям include с файлами HPP и lib с файлами bpi/lib, а также генерируются все файлы C++Builder, включая пакеты bpl и файлы библиотек libs.

 

 

 

10. Выполните компиляцию пакетов GLScene. Для этого щелкните правой кнопкой мыши в окне Project Manager по строке группового проекта GLSceneXE5 и выполните команду  Build All.  После компиляции установите в RAD Studio с помощью команды Install визуальные компоненты всех откомпилированных проектов, кроме GLS_SDL.bpl (версия SDL библиотеки не обновлена).

 

Установка компонентов завершена. Теперь можно работать с библиотекой GLScene. При выполнении описанных выше шагов инсталляции примеры для Delphi и C++Builder находятся в директории D:\Library\GLScene\Samples.
